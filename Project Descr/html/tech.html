Используемые технологии

Планируется использовать нестандартные технологии по написанию сайтов, в данном случае - корнем нашего веб-ресурса будет микрофреймворк Python, который называется Flask

  • Для оформления верстки и стилей будет использован фреймворк - Bootstrap
  • Backend будет поднят на микрофреймворке Python - Flask
  • В качестве веб-сервера будет использован Apache на Ubuntu server
  • SQL будет создан на модуле фреймворка SQLAlchemy
